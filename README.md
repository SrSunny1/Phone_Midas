# Phone_Midas
用自己手机摄像头当作便携式摄像头，使用局域网技术同步到电脑上，结合双线程帧队列技术来降低画面时延，并运行单目深度模型Midas   Use your own mobile phone camera as a portable camera, synchronize it to the computer using local area network (LAN) technology, reduce picture latency with dual-thread frame queue technology, and run the monocular depth model Midas.
